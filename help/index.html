B) Directories & File Uses

Our Design uses only assets & lib folder. So, don't touch any other folder for ui changes.

    • assets(Folder) - Put all Static images & other data (in Our Case images & json file) in this folder
    • lib(Folder) - Main folder that contains all the dart code
      • home_page(Folder) - Contain the start page and the page integrating the webview widget
      • on_boarding2(Folder) - Contain on_boarding2 page in this folder
      • on_boarding3(Folder) - Contain on_boarding3 page in this folder
      • tools(Folder) - Contain All tools widgets in this folder
        • admob_util file - Contain admob util and your id
        • tools_ad_banner file - Contain ad banner util
        • tools_theme file - Contain Theme util
      • app_state.dart - Contain all local states for your project.
    • main.dart - Entry Point of Our App.
  • pubspec.yaml(File) - Specification file (All the package and local assets (Like Images, Fonts, etc.) we use in this application is specify in this file.).

D) How to Create an Admob account

create an AdMob app:

  • Visit the AdMob homepage and signup using your Google account.
  • Go to the Dashboard, and click Apps on the left side menu.
  • Select the Platform as Android..
  • For the Is the app listed on a supported app store?, select Yes if your app is published on Playstore or Appstore.
  • Select No if your app is still under development.
  • Click on Continue.
  • Enter the App Name and click ADD APP.
  • Click Done.
  • Similarly, create another AdMob App for iOS app.

Adding key in your flutter project (Android)

  • First, get the APP key by visiting the AdMob dashboard -> Select the App -> App Settings (left side menu) -> Copy the App ID.
  • Paste App Id in web_companion/android/app/src/main/AndroidManifest.xml line number 61.

Adding key in your flutter project (ios)

  • First, get the APP key by visiting the AdMob dashboard -> Select the App -> App Settings (left side menu) -> Copy the App ID.
  • Paste App Id in web_companion\ios\Runner\Info.plist line number 69.

Assigning Ad Unit ID for Ad Banner

The Ad Unit ID is the unique identifier given to every ad on Admob. You can get this by creating a new ad unit from your Admob account.

To assign the ad unit Id for android app:

  • Go to the Admob account dashboard, click Apps on the left side menu and select your app.
  • Select Ad units and then click on the Add add unit.
  • Select the Banner Ad.
  • Enter the Ad unit name.
  • Click Create ad unit.
  • Copy the ad unit ID (the second one) and then click Done.
    • Paste it into the home page (web_companion\lib\home_page\home_page_widget.dart line 104
  • Similarly, create, copy and paste Ad unit for the iOS app as well.
